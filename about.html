About This Project

Project Overview

The Online Learning Management System (OLMS) is a web platform designed to help students access educational resources easily. This project was developed to demonstrate core web development skills.

Key Features

  • Secure Login: Students can create accounts and log in.
  • Course Dashboard: View video courses and descriptions.
  • Study Materials: Download PDF notes.
  • Management: Add, Edit, and Delete courses.

Technology Stack

HTML5 CSS3 PHP MySQL
